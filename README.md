# WebhookOps

Локально запускаемый шлюз надёжной доставки вебхуков: принимает внешние вебхуки, проверяет HMAC-подпись, дедуплицирует, сохраняет событие и асинхронно доставляет подписчикам с автоматическими повторами, DLQ и ручным повтором.

Модель надёжности: at-least-once + идемпотентность (без exactly-once).

> Статус: в разработке.

## Сервисы

- `ingest/` - FastAPI: горячий путь приёма вебхуков.
- `management/` - Django/DRF: модели, миграции, админка, воркеры доставки, публикатор outbox.
- `shared/` - общие контракты между сервисами.
